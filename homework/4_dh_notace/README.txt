Soubory
DH_97_scheme_x.fig je zadaný manipulátor jako Matlab 3D plot
DH_97_scheme_x.mat jsou parametry zadaného manipulátoru jako proměnná
DH_97_x.fig je výstup práce - 3D plot s doplněnýim souřadnými systémy
dh_x.txt jsou parametry DH notace pro dané manipulátory
dh_transform.m vrací transformační matici pro dané parametry DH notace
draw_coords.m přikreslí do 3D plotu souřadný systém daný transformační maticí
edit_figure_x.m je skript, kterým se programově upravuje DH_97_scheme.fig/mat pro získání DH_97_x.fig
